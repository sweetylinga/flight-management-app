# ✈️ Flight Booking Management App

A modern Flight Booking Management System built using Next.js, React, Tailwind CSS, and Zustand.

This project simulates a real-world airline booking flow with multi-step booking, seat selection, booking management, and persistent state management.

# 🚀 Features

## 🔍 Flight Search
- Search flights using:
  - Origin
  - Destination
  - Travel Date
  - Passenger Count

---

## ✈️ Flight Selection
- View available flights
- Flight cards display:
  - Airline
  - Route
  - Departure Time
  - Price
  - Class Type

---

## 💺 Seat Selection
- Interactive seat map
- Business & Economy classes
- Dynamic occupied seats
- Realtime-style seat updates
- Seat selection flow

---

## 🧾 Passenger Booking
- Passenger details form
- Passport Number
- Nationality
- Auto-generated PNR code

---

## ✅ Booking Confirmation
- Booking summary
- Flight details
- Seat details
- Passenger information
- Booking status

---

## 📂 My Bookings
- View all bookings
- Cancel booking
- Reschedule booking
- Persistent booking history

---

# ⚡ Zustand State Management

Implemented using:
- Zustand
- Persist middleware
- Partialize

Global state handles:
- Flight search
- Selected flight
- Selected seat
- Passenger details
- Booking history
- Current booking step

---

# 🔄 Realtime Seat Simulation

The app simulates realtime seat updates:
- Occupied seats disabled
- Dynamic UI updates
- Instant seat availability changes

## 1. Clone Repository
